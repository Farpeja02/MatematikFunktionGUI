# MatematikFunktionGUI

Dette projekt består af et program, som kan tage imod en matematisk funktion og deraf generere en graf. 

For at kunne køre dette program, skal du have installeret disse følgende pakker:
*sympy,*
*numpy,*
*matplotlib*
